Organix Landing Kit 2019

img-1.webp

Organix UI Kit is a modern and highly customizable web UI kit designed to enhance the speed and quality of web design projects. Built with compatibility and ease of use in mind, this kit provides a robust foundation for creating beautiful and functional websites. Designed for Sketch, it leverages the 12-column Bootstrap 4 grid system with a standard width of 1140px, ensuring responsiveness and alignment with contemporary web development practices.

Key Features

  1. 01 Pre-designed Components: Includes over 40 ready-to-use components and symbols to streamline the design process.
  2. 02 Landing Page Templates: Features 5 professionally crafted landing page templates, ideal for showcasing projects, services, or products.
  3. 03 Customizable: Fully organized and easy to modify, making it adaptable to various projects and branding requirements.
  4. 04 Compatible with Sketch: Seamlessly integrates with the popular design tool, allowing designers to work efficiently.
  5. 05 Responsive Design: Built with Bootstrap's grid system, ensuring a perfect fit for devices of all sizes.
